# RTS-Unit-Controller-Prototype

A small RTS prototype built in Unity featuring unit selection, movement, and combat mechanics.

## Scenes

- **BikeTestScene** — 1 Striker Bike + buildings
- **TankTestScene** — 1 Heavy Tank + buildings
- **Main** — 3 Tanks, 2 Bikes + buildings
